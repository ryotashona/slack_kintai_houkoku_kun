# Plans

Codexエージェントが作業する際は以下の手順で進捗を共有する。

1. plan作成: 取り組むタスクを2ステップ以上に分解し、Planツールへ登録する。
2. 作業: 各ステップを順番に実行し、重要な経過はこのリポジトリ上のPR/Issueコメントやチャットで共有する。
3. 作業履歴共有: ステップを完了したら、どのように対応したかを文章で報告する。
4. plan更新: `update_plan`を使ってステータスを同期し、次のステップへ進む。

補足:
- 単純作業（最も簡単なおよそ25%）のみPlanを省略してよい。
- Plan記録が必要なタスクでは、このディレクトリに補足資料やメモを追加してもよい。
